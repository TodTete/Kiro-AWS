# Spaces Astro

A 2D arcade space shooter built in Python + Pygame. The player pilots a spaceship, destroys asteroids and enemies, collects power-ups, and faces a boss at the end of each level.

## Core Features
- 8 selectable levels (planets), each with a unique background and boss
- 4 playable ship skins (Brayan, Fernanda, Marlin, Tete)
- 4 difficulty settings (Baja, Media, Alta, Extrema)
- Power-ups: speed boost (S), rapid fire (P), slow-motion (F)
- Bomb mechanic during boss fights
- Animated GIF backgrounds for menus and gameplay
- Persistent high score via `hiscore.txt`
- Dynamic camera with smooth panning
- Full audio system with per-channel volume control and mute toggle

## Language
Code comments, variable names, and UI strings are primarily in Spanish.
